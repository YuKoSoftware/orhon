1. **Default allocator** — explain that collections use SMP (GeneralPurposeAllocator) by default, no argument needed:
```

2. **Inline allocator** — pass allocator directly to .new():
```
import std::allocator
var arena: allocator.Arena = allocator.Arena.create()
defer { arena.deinit() }
var items: List(i32) = List(i32).new(arena.allocator())
```

```

var a = smp.allocator()
var items: List(i32) = List(i32).new(a)
